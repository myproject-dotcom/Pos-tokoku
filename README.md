
# POS Tokoku

Aplikasi POS berbasis web untuk penggunaan offline dan online.

## Fitur:
- Scan QR untuk input kode barang
- Form input nama, harga, jumlah
- Riwayat transaksi dalam tabel
- Edit dan hapus data per baris
